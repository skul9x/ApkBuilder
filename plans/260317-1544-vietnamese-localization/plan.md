## Overview
Dịch toàn bộ giao diện (UI) và các thông báo từ Go sang Tiếng Việt. Giữ nguyên Log RAW từ Gradle cho mục đích kỹ thuật nhưng dịch các nhãn log (Success/Error/Info). Chỉ hỗ trợ Tiếng Việt (No English).

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Go + Wails
- Config: JSON (`builder_settings.json`)
