# Phase 03: Frontend Localization

## Mục tiêu:
Dịch toàn bộ nhãn (Labels), Tiêu đề (Headers), Nút bấm (Buttons) và Thông báo (Tooltips) trong UI (`App.tsx`) sang Tiếng Việt.

## Các đầu việc:
- [ ] Dịch các nhãn chính: "Cấu hình", "Thư mục dự án", "Mẫu tên APK".
- [ ] Dịch các nút hành động: "Dọn dẹp", "Khởi chạy Build", "Cài đặt", "Dừng".
- [ ] Dịch các nhãn con: "Force Refresh", "Variant", "Devices".
- [ ] Dịch tiêu đề Console Terminal và các chú thích (Tooltip).
- [ ] Đảm bảo UI không bị vỡ/lệch khi chuỗi Tiếng Việt dài hơn.

## Các file sẽ chỉnh sửa:
- `frontend/src/App.tsx`

## Kiểm thử:
- [ ] UI hiển thị toàn bộ là Tiếng Việt.
- [ ] Layout không bị lỗi hoặc bị đè lên nhau.
